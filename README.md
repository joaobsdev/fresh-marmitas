🍱 Fresh Marmitas - Central de Pedidos

Sistema completo de gestão de pedidos para restaurante fit com cardápio digital, dashboard administrativo e interface moderna desenvolvida com React e TypeScript.

## 📋 Sobre o Projeto

A **Fresh Marmitas** é uma plataforma digital que conecta clientes a uma alimentação saudável com sabor de casa. O sistema conta com:

- 🛍️ **Cardápio digital** com categorias (PROTEICO, LOW CARB, LEVE, OMEGA 3)
- 📊 **Dashboard de gestão** para acompanhamento de pedidos em tempo real
- 🎨 **Interface moderna** com Tailwind CSS
- 🤖 **Imagens geradas por IA** para ilustrar os pratos
- ✅ **Fluxo completo de pedidos** (Pendentes → Em Preparo → Prontos → Entregues)

- ## 🚀 Funcionalidades

### Para o cliente
- Visualização do cardápio com categorias e filtros
- Adição de itens ao carrinho
- Resumo do pedido com cálculo automático de total
- Seleção de forma de pagamento (Na entrega / Cartão / Dinheiro)
- Campo de observações personalizadas

### Para o gestor (Dashboard)
- Visualização de pedidos organizados por status:
  - ⏳ Pendentes
  - 🍳 Em Preparo
  - ✅ Prontos
  - 📦 Entregues
- Detalhamento de cada pedido (itens, quantidade, total)
- Informações do cliente (nome, telefone, horário do pedido)
- Atualização de status em tempo real

- ## 🛠️ Tecnologias Utilizadas

- **React**
- **Next.JS**
- **TypeScript**
- **Tailwind CSS**
- **Context API**

- Deploy: https://fresh-marmitas.vercel.app/
